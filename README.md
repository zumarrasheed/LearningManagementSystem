# Learning Management System (Database - SQL)

A relational database-based Learning Management System (LMS) designed using SQL.
This project manages and tracks data related to students, teachers, courses, and university operations while demonstrating relationships between different entities.

## Description

The system is designed to store, organize, and manage academic data efficiently.
It models real-world university operations by establishing relationships between students, teachers, and courses using relational database concepts.

## Features

* Student management
* Teacher management
* Course handling
* University data organization
* Relationship mapping between entities
* Query-based data retrieval

## Technologies Used

* SQL
* Relational Database Concepts (ER Model, Normalization, Joins)

## Database Design

* Entities: Students, Teachers, Courses, Departments
* Relationships:

  * Students enroll in courses
  * Teachers teach courses
  * Courses belong to departments

## 🚀 How to Run

1. Open any SQL environment:

   * MySQL / SQL Server / Oracle

2. Import or run the SQL script file:
   database.sql

3. Execute queries to interact with the system
